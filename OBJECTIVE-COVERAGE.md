# AZ-900 Objective Coverage

This project includes one or more flashcards for each bullet-level objective from the AZ-900 objectives PDF marked "Skills measured as of October 30, 2025."

## 1. Describe Cloud Concepts

### OBJ 1.1 - Describe cloud computing

- Define cloud computing
- Describe the shared responsibility model
- Define public, private, and hybrid cloud models
- Identify use cases for each cloud model
- Describe the consumption-based model
- Compare cloud pricing models
- Describe serverless

### OBJ 1.2 - Describe the benefits of using cloud services

- High availability and scalability
- Reliability and predictability
- Security and governance
- Manageability

### OBJ 1.3 - Describe cloud service types

- Infrastructure as a service (IaaS)
- Platform as a service (PaaS)
- Software as a service (SaaS)
- Use cases for IaaS, PaaS, and SaaS

## 2. Describe Azure Architecture and Services

### OBJ 2.1 - Describe core Azure architectural components

- Regions, region pairs, and sovereign regions
- Availability zones
- Azure datacenters
- Azure resources and resource groups
- Subscriptions
- Management groups
- Resource group, subscription, and management group hierarchy

### OBJ 2.2 - Describe Azure compute and networking services

- Containers, virtual machines, and functions
- Azure VM options, VM Scale Sets, availability sets, and Azure Virtual Desktop
- Resources required for virtual machines
- Application hosting options
- Virtual networks, subnets, peering, Azure DNS, VPN Gateway, and ExpressRoute
- Public and private endpoints

### OBJ 2.3 - Describe Azure storage services

- Azure Storage services
- Storage tiers
- Redundancy options
- Storage account options and storage types
- File movement tools such as AzCopy, Storage Explorer, and Azure File Sync
- Migration tools such as Azure Migrate and Azure Data Box

### OBJ 2.4 - Describe Azure identity, access, and security

- Microsoft Entra ID and Microsoft Entra Domain Services
- Single sign-on, multifactor authentication, and passwordless authentication
- External identities including B2B and B2C
- Microsoft Entra Conditional Access
- Azure role-based access control
- Zero Trust
- Defense in depth
- Microsoft Defender for Cloud

## 3. Describe Azure Management and Governance

### OBJ 3.1 - Describe cost management in Azure

- Factors that affect Azure costs
- Azure Pricing Calculator
- Microsoft Cost Management capabilities
- Tags

### OBJ 3.2 - Describe governance and compliance tools

- Microsoft Purview
- Azure Policy
- Resource locks

### OBJ 3.3 - Describe management and deployment tools

- Azure portal
- Azure Cloud Shell, Azure CLI, and Azure PowerShell
- Azure Arc
- Infrastructure as code
- Azure Resource Manager and ARM templates

### OBJ 3.4 - Describe monitoring tools

- Azure Advisor
- Azure Service Health
- Azure Monitor, Log Analytics, alerts, and Application Insights

# AZ-900 Flashcards on Azure

A simple flashcard study app for Microsoft Azure Fundamentals: AZ-900.

This project is designed to be hosted on Azure Static Web Apps and used as a practical way to connect AZ-900 exam concepts with a real Azure deployment.

## Features

- Flip cards between question and answer
- Filter by AZ-900 category
- Shuffle cards
- Track known and missed cards
- Works as a static site with HTML, CSS, and JavaScript

## AZ-900 Concepts Demonstrated

- Public cloud hosting
- Managed/PaaS-style hosting
- Azure Static Web Apps
- GitHub-based CI/CD
- Resource groups
- Azure regions
- HTTPS
- Cost management using a free tier
- Shared responsibility model

## Suggested Azure Deployment

1. Push this repo to GitHub.
2. In the Azure portal, create an Azure Static Web App.
3. Connect the Static Web App to this GitHub repository.
4. Set the app location to `/`.
5. Leave the output location blank for this plain HTML/CSS/JS project.
6. Let Azure create the GitHub Actions workflow.

## Why I Built This

I built this while studying for AZ-900 so I could turn Azure concepts into flashcards and practice deploying a simple app with Azure Static Web Apps.
